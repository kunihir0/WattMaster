For Non-Jailbroken Users
  1. Install TrollStore on your device.
  2. Install TrollFools with TrollStore.
  3. Download the latest .deb file from our Releases page.
  4. Open TrollFools then Select Wattpad
  5. Inject our .deb packge into Wattpad
  6. Profit!

Supported versions for TrollStore/TrollFools: 14.0 beta 2 - 16.6.1, 16.7 RC (20H18), 17.0
